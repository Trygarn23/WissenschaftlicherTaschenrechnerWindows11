Wissenschaftlicher Taschenrechner (Windows 11)

Dieses Projekt ist ein selbst entwickelter wissenschaftlicher Taschenrechner in Java mit grafischer Oberfläche (Swing).
Entstanden ist das Ganze aus purer Neugier und ein bisschen Langeweile – nicht, weil es von der Schule verlangt wurde, sondern weil ich ausprobieren wollte, wie weit man über den Unterricht hinausgehen kann.

Funktionen:
Grundrechenarten
Wissenschaftliche Funktionen wie Potenzen, Klammern und Prioritäten
Eigener mathematischer Parser
Grafische Benutzeroberfläche mit Java Swing
Eingabe über Buttons und Tastatur

Motivation:
Ziel war es, keinen 0815-Schul-Taschenrechner zu bauen, sondern etwas, das:
über das normale Schulniveau hinausgeht
saubere Trennung von Logik und UI nutzt
mathematische Ausdrücke selbst verarbeitet

Technisches
Sprache: Java
GUI: Swing
IDE: IntelliJ IDEA
Plattform: Windows 11

Hinweis
Dieses Projekt dient Lern- und Übungszwecken und ist öffentlich einsehbar, um den Code nachvollziehen zu können.

Lizenz
Dieses Projekt ist öffentlich einsehbar, der Quellcode darf jedoch ohne ausdrückliche Zustimmung nicht kopiert, weiterverwendet oder als eigenes Projekt ausgegeben werden.
